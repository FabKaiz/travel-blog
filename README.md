# Travel Blog ✈️⛰️🏝️
### This travel blog was made using: 
  * Sanity for the backend and CMS
  * NextJs for the frontend
  * mapboxGL to show the emplacement of the blogger on the map
  * groq for the backend query
  * NProgress for the loading bar
